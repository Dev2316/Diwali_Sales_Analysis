# 🎆 Diwali Sales Analysis Dashboard (Group - 9)

This project presents an **interactive Power BI dashboard** created using **Diwali Sales Data**. The goal of this analysis is to understand customer purchasing behavior during the Diwali festival season and identify key sales trends based on **gender, age group, and marital status**.

## 📊 Dashboard Overview

The dashboard provides a visual summary of:

* **Total Orders:** 11K
* **Total Sales:** 106.25M
* **Sum of Amount by Gender** (Pie Chart)
* **Count of Orders by Gender** (Pie Chart)
* **Count of Gender by Age Group** (Bar Chart)
* **Count of Marital Status by Age Group and Gender** (Line Chart)

It also includes a **Product Category filter**, allowing users to explore data for specific product segments.

## 🧠 Key Insights

* **Males** contribute a higher share of total sales and orders compared to females.
* The **26–35 age group** is the most active segment in purchasing during Diwali.
* Married individuals tend to make more purchases, especially within the 26–45 age range.
* The dashboard provides a detailed breakdown of **consumer demographics and spending behavior**.

## ⚙️ Tools & Techniques Used

* **Power BI** for visualization and dashboard design
* **Data cleaning and preprocessing** using Power Query
* **DAX measures** for aggregations and insights
* **Interactive visuals** including pie charts, bar charts, line charts, and slicers

## 🎯 Project Objective

The main objective of this project was to **analyze Diwali sales performance** and discover patterns in customer demographics that influence purchasing decisions. This insight can help in **targeted marketing strategies** and **improving sales outcomes** during festive seasons.
